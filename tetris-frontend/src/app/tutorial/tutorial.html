→ (Direita): move a peça para a direção indicada


← (Esquerda): move a peça para a direção indicada


↓ (Baixo): Move a peça para baixo mais rápido



O Tetris é um jogo de quebra-cabeça onde peças geométricas (tetraminós) caem

continuamente do topo da tela. O objetivo é girar e encaixar essas peças para

formar linhas horizontais completas. Quando uma linha é formada, ela desaparece,

impedindo que os blocos acumulem até o topo da tela
